## Radio

Используется для выбора одного варианта из нескольких взаимоисключающих опций в рамках одной группы

<hr />

### Свойства

#### disabled?: boolean
Переключает интерактивное состояние компонента - доступное или недоступное для взаимодействия

#### isChecked: boolean
Переключает выбранное/не выбранное состояние

#### isError?: boolean
Переключает компонент в состояние визуализации ошибки ввода

<hr />

### Кастомизация

#### iconChecked: ReactElement<'IconProps>
Компонент иконки при выбранном состоянии

#### iconEmpty: ReactElement<'IconProps>
Компонент иконки при не выбранном состоянии

#### styles?: StyleProp<'ViewStyle>
Стили для контейнера кнопки
