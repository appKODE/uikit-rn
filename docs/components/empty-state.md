## EmptyState

Используется для обозначения пустого состояния. В случаях, когда контент, который должен быть на экране, по каким-то причинам отсутствует - не добавлине пользователями, удалён пользователями, пришео пустой ответ от сервера и тд.

### Свойства

##### titleText: string
Заголовок, всегда обязателен

#### iconInstance?: FC<'IconProps>
Иконка, опциональна

#### messageText?: string
Дополнительный текст под заголовком, опицонален

#### iconSize?: number
Размер ионки, опицонален (по дефолту 56x56px)

#### iconColor?: IconColorKeys
Цвет иконки, опицонален (по дефолту iconQuaternary)

#### titleColor?: TextColorKeys
Цвет текста заголовка, опицонален (по дефолту textTertiary)

#### messageColor?: TextColorKeys
Цвет текста сообщения, опицонален (по дефолту textTertiary)
